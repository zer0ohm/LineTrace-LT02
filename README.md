# linetrace-lt02
## 概要
このリポジトリは、ライントレース「LT02m」の成果物を公開するものです。

## 内容 (予定)
 - 回路図(.sch)
 - PCB設計データ(.kicad_pcb)
 - ガーバーデータ (Fusion PCB準拠)
 - その他諸々回路に必要なファイル
 - 制御ソースコード
